# a blog

### fork
fork this and replace the _post directory with your stuff

also of interest would probably be assets/, _includes/, _layouts/

## local dev

  ./bin/dev

open localhost:4000 in your browser

## deploy

set git remote to your [user].github.io repo.
  
  git push
